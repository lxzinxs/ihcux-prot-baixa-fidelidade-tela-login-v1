# 📱 Protótipo de Interface: Fluxo de Autenticação Móvel

Este repositório apresenta um exercício prático de **Interação Humano-Computador (IHC)** e **Design de Experiência do Usuário (UX)**, focado na criação de um wireframe de baixa fidelidade para dispositivos móveis.

---

## 🔍 Visão Geral do Exercício

O objetivo desta atividade foi estruturar a hierarquia visual de uma tela de login, garantindo que os elementos essenciais para a jornada do usuário estivessem posicionados de forma intuitiva e funcional. O protótipo foca na simplicidade para reduzir a carga cognitiva durante o acesso ao sistema.

---

## 🏗️ Elementos do Protótipo

A interface foi desenhada contendo os seguintes componentes fundamentais:

* **Avatar de Perfil**: Localizado no topo para confirmar visualmente a finalidade da tela de identificação.
* **Campo de Entrada "Username"**: Espaço dedicado para a inserção da identificação única do utilizador.
* **Campo de Entrada "Senha"**: Local para inserção da credencial de segurança.
* **Botão de Ação**: Elemento de interação principal (Call to Action) para processar o formulário e realizar o acesso.

---

## 🛠️ Ferramentas e Conceitos Aplicados

* **Low-Fidelity Wireframing**: Foco na estrutura e funcionalidade em vez de detalhes visuais finais.
* **Arquitetura de Informação**: Disposição vertical dos elementos para facilitar a leitura em telas móveis.
* **Princípios de IHC**: Aplicação de visibilidade nos campos de entrada e botões de interação.
